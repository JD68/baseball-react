# Personal baseball project using react.

Real application with real baseball history database! 

# Dependencies

- Node.js
- NPM

# Install and run instructions

```sh
$ npm install
```

Start the local dev server:

```sh
$ npm run server
```

Navigate to **http://localhost:8080/** to view the app.
